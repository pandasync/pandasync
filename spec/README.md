# PandaSync Protocol Specification

This directory contains the PandaSync protocol specification.

## Current Version

- [v0.1](v0.1/) -- Draft specification (in progress)

## Structure

Each version contains five chapters covering the full protocol stack:

1. **Introduction** -- scope, terminology, conformance
2. **Transport** -- RTP, QUIC, WebRTC wire formats
3. **Clock Sync** -- PTP profiles, auto-configuration, fallbacks
4. **Discovery** -- mDNS, unicast DNS-SD, cloud registry
5. **Control Plane** -- REST API, WebSocket events, NMOS compatibility
